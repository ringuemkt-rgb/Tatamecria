# Capoeira de Cria — Status do projeto

Este repositório foi preparado como ponto de entrada no GitHub para o projeto do jogo.

## O que já existe no material consolidado
- base integrada do jogo
- protótipo web
- shell Android
- sistemas de IA
- mundo aberto com ciclo de dia/noite
- biblioteca visual e bíblia visual
- sistema de prompts
- documentação de produção

## O que ainda exige integração final real
- animações finais frame a frame
- áudio final completo
- balance final
- QA em dispositivos reais
- APK/AAB final assinado

## Uso recomendado
1. importar a base integrada do projeto
2. abrir a pasta Android no Android Studio
3. validar o protótipo web
4. integrar assets finais e polir o loop jogável
