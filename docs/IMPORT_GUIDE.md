# Importação da base do jogo

## Estrutura recomendada
- `app/web/` → protótipo web
- `app/android/` → shell Android
- `assets/` → biblioteca visual e recursos Android-ready
- `data/` → sistemas, IA, mundo aberto e manifestos
- `docs/` → documentação consolidada
- `prompts/` → sistema oficial de prompts visuais

## Fluxo de trabalho recomendado
1. importar a base integrada do projeto para este repositório
2. validar `app/web/index.html`
3. abrir `app/android/` no Android Studio
4. integrar sistemas recentes de IA, mundo aberto e combate
5. usar a documentação visual e os packs de referência para fechar assets finais

## Observação honesta
Este repositório no GitHub não significa, por si só, que o jogo final comercial já esteja concluído.
